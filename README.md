# 🚀 AI Resume Job Matcher

An AI-powered web application that analyzes resumes (PDF) and compares them with job descriptions to provide actionable insights.

## 📌 Features

- Upload Resume (PDF)
- Paste Job Description
- AI-generated Match Score (0–100)
- Matching Skills Detection
- Missing Skills Identification
- Improvement Suggestions
- Visual Skill Comparison Chart

---

## 🧠 Tech Stack

- Python
- Streamlit
- PyPDF2 (PDF parsing)
- OpenRouter API (LLM)
- Pandas (data visualization)

---

## ⚙️ How It Works

1. User uploads resume (PDF)
2. Resume text is extracted using PyPDF2
3. Job description is provided by user
4. AI compares resume with job description
5. Returns:
   - Match Score
   - Matching Skills
   - Missing Skills
   - Suggestions
